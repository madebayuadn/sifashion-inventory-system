Sistem Manajemen Inventaris Produk Fashion (SiFashion)

Deskripsi:

SiFashion adalah aplikasi untuk mengelola stok pakaian dan aksesori secara terorganisir. Data utama yang digunakan adalah data produk, data kategori ukuran, dan data varian warna. Pengguna aplikasi adalah pengelola toko atau admin gudang fashion.

Spesifikasi:

a. Pengguna dapat menambahkan, mengubah, dan menghapus data produk fashion.

b. Sistem dapat mencatat detail stok barang berdasarkan kombinasi ukuran dan warna tertentu.

c. Pengguna dapat mencari data produk berdasarkan kriteria ukuran atau warna menggunakan Sequential dan Binary Search.

d. Pengguna dapat mengurutkan data produk berdasarkan harga atau jumlah stok menggunakan Selection dan Insertion Sort.

e. Sistem dapat menampilkan statistik jumlah produk yang paling populer dan sisa total stok per kategori.
